# 🎀 Sona DBMS - Управление базами данных

**Sona DBMS** - это веб-приложение для управления базами данных PostgreSQL с красивым розовым интерфейсом. Позволяет создавать таблицы, добавлять/редактировать/удалять записи, выполнять SQL запросы через удобный интерфейс.

## ✨ Возможности

### 🗄️ Управление базами данных
- Подключение к нескольким серверам PostgreSQL
- Создание новых баз данных прямо из интерфейса
- Переключение между базами данных
- Сохранение подключений

### 📊 Управление таблицами
- Создание таблиц с выбором типов данных
- Просмотр всех таблиц в базе данных
- Удаление таблиц с подтверждением
- Автоматическое определение структуры таблиц

### 📝 Работа с данными
- Добавление новых записей через форму
- Редактирование существующих записей (двойной клик)
- Удаление записей
- Копирование/дублирование записей

### 💻 SQL Редактор
- Выполнение любых SQL запросов
- История запросов
- Сохранение часто используемых запросов

### 🎨 Интерфейс
- Розовый дизайн с градиентами
- Адаптивная верстка
- Удобные модальные окна
- Всплывающие уведомления

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/sommylvd/SonaDBMS.git
cd SonaDBMS
```

### 2. Настройка базы данных PostgreSQL

Запустите PostgreSQL и создайте базу данных:

```sql
CREATE DATABASE sonya_db;
```

### 3. Настройка бэкенда

```bash
python main.py
```

**Ожидаемый вывод:**
INFO: Uvicorn running on http://127.0.0.1:8000
INFO: Application startup complete.


### 5. Настройка фронтенда

*Откройте НОВЫЙ терминал!*

```bash
# Перейдите в папку фронтенда
cd frontend

# Установите зависимости
npm install

# Запустите приложение
npm start
```

### 6. Открытие приложения

Перейдите в браузере: [http://localhost:3000](http://localhost:3000/)

## 🔌 Подключение к базе данных

1. Нажмите **CONNECT**
2. Нажмите **+ ADD DATABASE**
3. Заполните поля:

| Поле | Значение |
|------|----------|
| Connection Name | `Local` (любое имя) |
| Host | `localhost` |
| Port | `5432` |
| Database Name | `sonya_db` |
| Username | `postgres` |
| Password | ваш пароль |

4. Нажмите **ADD**
5. Выберите созданную БД из списка
